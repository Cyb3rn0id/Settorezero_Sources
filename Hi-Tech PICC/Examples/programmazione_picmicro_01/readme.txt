Questi file sono di supporto alla terza lezione
del corso di programmazione dei picmicro in C
disponibile sul sito www.settorezero.com e sono
scaricabili unicamente dagli utenti iscritti al sito.

sorgenti\main.c => programma principale
sorgenti\delay.c => funzioni di ritardo
sorgenti\delay.h => file header necessario a delay.c
eseguibile\lampeggia_led.hex => programma già compilato per pic16f877A

Seguire la lezione e lo schema elettrico per capirne
il funzionamento.
