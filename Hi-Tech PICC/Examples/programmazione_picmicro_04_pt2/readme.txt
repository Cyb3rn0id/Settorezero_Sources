Questi file sono di supporto alla settima lezione
del corso di programmazione dei picmicro in C
disponibile sul sito www.settorezero.com e sono
scaricabili unicamente dagli utenti iscritti al sito.

\caratteri_custom
	\eseguibile
		\lcdcustomchars.hex => programma già compilato per pic16f877A (checksum: 68D9)
	\sorgenti
		\main.c => programma principale
		\settings.h => impostazioni di configurazione per programma principale
		\delay.c => funzioni di ritardo
		\delay.h => file header necessario a delay.c
		\lcd.c => routine per gestione LCD
